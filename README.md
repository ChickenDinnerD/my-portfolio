# Portfolio Website

Personal portfolio website built with React + Vite.

## Stack

* React
* Vite
* JavaScript
* React Router DOM
* Tailwind CSS
* Framer Motion
* React Icons
* Vercel (deployment)

## Features

* Multi-page portfolio
* Responsive design (mobile / tablet / desktop)
* Projects showcase
* Project detail pages
* About page
* Contact page
* Resume download
* Social links (GitHub / LinkedIn / Telegram / Email)
* Smooth animations
* Clean modern UI
* Fast deployment with Vercel

## Pages

* Home
* About
* Projects
* Project Details
* Contact
* Resume
* 404 Page

## Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
